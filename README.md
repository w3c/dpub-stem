dpub-stem
=========

Documents produced by the [STEM task force](https://www.w3.org/dpub/IG/wiki/Task_Forces/Accessibility) of the [Digital Publishing Interest Group](http://www.w3.org/dpub/IG) of the W3C.

See also a [paged view](http://w3c.github.io/dpub-stem/) of the document served in HTML.

If you are member of the interest group, and you wish to contribute to the content of this repo, please contact Ivan Herman (<ivan@w3.org>) or Thierry Michel (<tmichel@w3.org>), giving them your github login.
